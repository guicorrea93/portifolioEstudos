# portifolioEstudos

Portfólio de estudos práticos realizados em cursos e plataformas de ensino.

Cada estudo possui uma página interativa com análises, gráficos e código completo.

**Acesse:** [guicorrea93.github.io/portifolioEstudos](https://guicorrea93.github.io/portifolioEstudos/)

## Estrutura

Cada estudo segue o padrão `estudo_XX/` contendo:

- `index.html` — Página interativa com visualizações e análises
- `README.md` — Documentação do estudo
- `thumbnail.png` — Imagem de preview
- Notebooks Jupyter (`.ipynb`) — Código fonte
